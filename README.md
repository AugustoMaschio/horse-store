# Horse Store

Landing page responsiva de uma loja virtual de arreios, desenvolvida com HTML, CSS e jQuery.

## Funcionalidades
- Layout responsivo para desktop, tablet e mobile
- Menu dropdown no desktop
- Menu hambúrguer no mobile
- Grid de produtos com efeito hover
- Estrutura semântica com HTML5

## Tecnologias
- HTML5
- CSS3
- JavaScript
- jQuery
